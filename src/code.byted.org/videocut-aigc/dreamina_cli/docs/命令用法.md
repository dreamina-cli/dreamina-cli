# Dreamina CLI 命令用法

本文基于当前 `dreamina help` 及各子命令 `-h` 内容整理，面向日常使用场景，统一用中文说明主要命令、适用场景和常见调用方式。

## 快速开始

推荐按下面顺序使用：

1. 先执行 `dreamina login` 完成本地登录。
2. 再执行生成命令，例如 `dreamina text2image --prompt="一只猫的肖像"`。
3. 遇到异步任务时，用 `dreamina query_result --submit_id=<任务ID>` 查询结果，或用 `dreamina list_task` 查看历史任务。
4. 使用 `dreamina user_credit` 查看当前账号剩余积分。

如果是 AI、Agent 或远程环境驱动登录，优先使用：

```bash
dreamina login --headless
```

或：

```bash
dreamina relogin --headless
```

Linux 下需要预先安装可执行名为 `google-chrome` 的 Google Chrome。

## 顶层命令分类

顶层可见命令分为两类。

账号与基础命令：

- `login`：登录并保存本地登录态
- `relogin`：清理旧登录态后重新登录
- `import_login_response`：把浏览器登录返回的完整 JSON 导入本地凭证
- `query_result`：按 `submit_id` 查询单个异步任务结果
- `list_task`：查看本地保存的任务列表
- `user_credit`：查询当前账号剩余积分
- `logout`：清除本地登录态
- `version`：查看版本与提交信息
- `help`：查看命令帮助

生成类命令：

- `text2image`：文生图
- `image2image`：图生图
- `image_upscale`：图片超分
- `text2video`：文生视频
- `image2video`：单图生视频
- `frames2video`：首尾帧生视频
- `multiframe2video`：多图连续故事视频
- `multimodal2video`：多模态视频生成

补充说明：

- `completion` 是内置补全命令，用于生成 shell 自动补全脚本
- `ref2video` 是兼容旧名称的别名命令，已提示迁移到 `multiframe2video`
- `validate-auth-token` 属于调试命令，不会出现在顶层主帮助摘要中

## 登录与账号命令

### `login`

用途：

- 复用当前本地登录态
- 登录态失效时自动拉起浏览器继续登录
- 在 `--headless` 模式下通过无头浏览器展示登录二维码

常用示例：

```bash
dreamina login
dreamina login --headless
dreamina login --debug
```

### `relogin`

用途：

- 先删除本地登录态，再强制重新登录

常用示例：

```bash
dreamina relogin
dreamina relogin --headless
dreamina relogin --debug
```

### `import_login_response`

适用场景：

- 浏览器登录在另一台机器上完成
- Agent 帮你打开登录页，但最终需要把完整 JSON 导回本机 CLI

常用示例：

```bash
dreamina import_login_response --file /tmp/dreamina-login.json
cat /tmp/dreamina-login.json | dreamina import_login_response
```

### `user_credit`

用途：

- 查询当前登录账号的剩余积分

示例：

```bash
dreamina user_credit
```

### `logout`

用途：

- 删除本地登录凭证，不影响任务数据和配置

示例：

```bash
dreamina logout
```

## 任务查询命令

### `query_result`

用途：

- 按 `submit_id` 查询某个异步任务当前状态和结果

常用参数：

- `--submit_id`：任务 ID
- `--download_dir`：把结果媒体文件下载到指定目录

示例：

```bash
dreamina query_result --submit_id=3f6eb41f425d23a3
```

### `list_task`

用途：

- 查看当前登录用户保存到本地任务库中的任务列表

常用参数：

- `--gen_status`：按任务状态过滤
- `--gen_task_type`：按任务类型过滤
- `--submit_id`：按任务 ID 过滤
- `--limit`：返回数量上限，默认 `20`
- `--offset`：分页偏移

示例：

```bash
dreamina list_task
dreamina list_task --gen_status=success
```

## 生成类命令

所有生成命令都会消耗积分。多数命令是异步任务，可以配合 `--poll` 在提交后短轮询，超时再回退到 `query_result`。

### `text2image`

用途：

- 直接根据文本生成图片

特点：

- 支持 `3.0`、`3.1`、`4.0`、`4.1`、`4.5`、`4.6`、`5.0`、`lab`
- `lab` 仅限 VIP
- 常见比例：`1:1`、`16:9`、`9:16` 等

示例：

```bash
dreamina text2image --prompt="a cat portrait" --ratio=1:1 --resolution_type=2k
```

### `image2image`

用途：

- 上传一张或多张本地图像做重绘或风格改写

示例：

```bash
dreamina image2image --images ./input.png --prompt="turn into watercolor"
```

### `image_upscale`

用途：

- 对单张图片做超分

说明：

- `2k` 对非 VIP 可用
- `4k`、`8k` 需要 VIP

示例：

```bash
dreamina image_upscale --image=./input.png --resolution_type=4k
```

### `text2video`

用途：

- 直接根据文本生成视频

说明：

- 默认模型为 `seedance2.0fast`
- 当前已支持 `seedance2.0_vip` 与 `seedance2.0fast_vip`
- 支持比例 `1:1`、`16:9`、`9:16` 等
- `seedance2.0` 家族当前为 `720p`

示例：

```bash
dreamina text2video --prompt="a cat running" --duration=5
```

### `image2video`

用途：

- 以单张图片作为起始帧生成视频

说明：

- 适合单图动画
- 多图故事推荐用 `multiframe2video`
- 全参考混合输入推荐用 `multimodal2video`

示例：

```bash
dreamina image2video --image=./first.png --prompt="camera push in"
```

### `frames2video`

用途：

- 上传首帧和尾帧生成过渡视频

示例：

```bash
dreamina frames2video --first=./start.png --last=./end.png --prompt="season changes"
```

### `multiframe2video`

用途：

- 上传 2 到 20 张图片，生成连续叙事视频

说明：

- 2 张图时可直接用 `--prompt` 和可选 `--duration`
- 3 张及以上建议逐段传 `--transition-prompt`
- 每段时长支持 `[0.5, 8]` 秒，总时长需不小于 `2` 秒

示例：

```bash
dreamina multiframe2video --images ./a.png,./b.png --prompt="character turns around"
dreamina multiframe2video --images ./a.png,./b.png,./c.png --transition-prompt="turn from A to B" --transition-prompt="turn from B to C"
```

### `ref2video`

用途：

- 兼容旧版本命令名

说明：

- 当前会提示该命令已废弃，建议改用 `multiframe2video`
- 参数与 `multiframe2video` 保持兼容

示例：

```bash
dreamina ref2video --images ./a.png,./b.png --prompt="character turns around"
```

### `multimodal2video`

用途：

- 混合图片、视频、音频输入，执行更强的多模态视频生成

说明：

- 至少要有一个 `--image` 或 `--video`
- 纯音频输入不支持
- 支持 `seedance2.0`、`seedance2.0fast`、`seedance2.0_vip`、`seedance2.0fast_vip`

示例：

```bash
dreamina multimodal2video --image ./input.png --prompt="turn this into a cinematic shot"
dreamina multimodal2video --image ./input.png --audio ./music.mp3 --model_version=seedance2.0fast --duration=5
dreamina multimodal2video --image ./input.png --video ./ref.mp4 --audio ./music.mp3 --model_version=seedance2.0fast --duration=5
```

## 自动补全命令

### `completion`

用途：

- 输出 shell 自动补全脚本

支持目标：

- `bash`
- `zsh`
- `fish`
- `powershell`

查看帮助：

```bash
dreamina help completion
```

示例：

```bash
dreamina completion bash
dreamina completion zsh
dreamina completion fish
dreamina completion powershell
```

## 调试命令

### `validate-auth-token`

用途：

- 调试本地登录凭证是否能被后端接受

示例：

```bash
dreamina validate-auth-token
```

## 常见提示

- 长 JSON 登录响应不建议直接贴到聊天频道，容易被截断，优先用本地终端粘贴或写入文件后再导入。
- `Seedance 2.0` 系列是当前重点视频模型族，质量优先时可以优先考虑。
- 某些高内容安全风险模型首次使用前，可能需要先在 Dreamina Web 端完成授权确认。
- 遇到完整参数说明时，优先执行 `dreamina <子命令> -h` 查看最新帮助。
